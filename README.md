# GDUT校园网一键拨号器 (GDUT Wifi Connector)

## 1. 简介
本工具旨在提供优于学校官方客户端的、更加简洁优雅的宽带拨号连接体验。 

程序具备**自动检测与创建连接**、**账号密码本地记忆**等功能，具备**基础的图形化UI**。底层直接调用 Windows API 进行拨号，剥离了所有冗余组件，连接速度更快，随开随用。 

本程序的底层逻辑适用于所有标准的 PPPoE 拨号上网环境，欢迎其他学校的同学下载使用。

**✨ v1.1.0 核心升级：**
* **自动准备环境**：程序运行前会自动检测系统的拨号配置，若未配置则会在后台自动创建。
* **全自动无感拨号**：首次保存账密后，再次打开程序将自动读取、自动执行拨号，并在连接成功 3 秒后自动关闭窗口。

## 2. 对于使用者
无需繁琐的安装配置，开箱即用：

1. **下载程序**：在页面右侧找到 **Releases**，下载最新版本的 `.exe` 程序文件。
2. **初次运行**：双击打开程序（若系统拦截，请点击“更多信息” -> “仍要运行”），输入你的校园网账号与密码，点击“开始拨号”。
3. **日常使用 (全自动)**：程序会自动记忆账密。下次需要连网时，直接双击运行程序，它将全自动完成拨号并退出，无需任何手动点击！

💡 **进阶：如何设置开机无感自动连网？**
结合 v1.1.0 的全自动特性，您可以这样设置开机自动拨号：
1. 找到您下载的 `.exe` 程序，右键点击并选择“创建快捷方式”。
2. 按下键盘的 `Win + R` 键打开“运行”窗口，输入 `shell:startup` 并回车，此时会弹出一个“启动”文件夹。
3. 把刚才创建的**快捷方式**拖拽（或复制）到这个文件夹中。大功告成！之后每次开机，电脑都会自动帮您连接网络。

## 3. 对于开发者
本工具基于 Python 和 Tkinter 原生库开发。如果您想从源码运行或进行二次开发：
1. 克隆本仓库到本地。
2. 确保系统已配置 Python 3.x 环境。
3. 在终端中运行 `python main.py` 即可启动。
